


## Task # 2: Website Designing Task
Note: You may use AI for assistance
You have been hired to design a website for a new coffee shop called "Java Buzz". The coffee shop wants
a modern and inviting website that showcases their coffee, menu, and location. They also want to be
able to take online orders and have a blog section to share news and updates.
## Requirements:
The website should have a simple and intuitive navigation bar at the top.
The homepage should feature high-quality images of their coffee, along with a prominent call-to-action
button to place an order.
The menu section should be easy to read and navigate, with prices clearly displayed.
The location section should include a map and address, as well as directions and parking
information.(using leaflet API)
The blog section should be dynamic using JSON Data and organized layout that's easy to read and share
on social media.(Atleast 10 blogs)
The website should be responsive and work well on a desktop, tablet, and mobile devices.
The color scheme should be warm and inviting, with shades of brown and green.
To create site map.xml file using online tools.
Create robot.text file , also add title, keywords, description in meta tags.

In submission file push your all website file in Github repository and paste the rep link in readme.md file

## Judging Criteria:
Creativity and originality of the design.
User-friendliness and ease of navigation.
Use of color, typography, and images.
Overall visual appeal and cohesiveness.
Responsiveness and compatibility across different devices.
