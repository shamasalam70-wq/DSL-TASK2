# Java Buzz - Website Designing Task

Java Buzz is a modern and inviting coffee shop website designed as part of Task 2.

## Features
- **Modern Navigation**: A responsive, sticky navigation bar.
- **Hero Section**: High-quality coffee imagery with a clear call-to-action.
- **Dynamic Menu**: A cleanly laid out menu with categories and prices, powered by JSON.
- **Location & Map**: Integrated Leaflet API for location mapping, including directions and parking information.
- **Dynamic Blog**: A grid-based blog section pulling from a JSON dataset of 10+ entries.
- **Responsive Design**: Fully optimized for Desktop, Tablet, and Mobile devices.
- **Aesthetic**: A warm, inviting color scheme using shades of brown and green.
- **SEO Optimized**: Includes `robots.txt`, `sitemap.xml`, and meta tags for improved search engine visibility.

## Technologies Used
- **React (TypeScript)**
- **Vite** (for fast development and building)
- **Vanilla CSS** (for precise, custom styling)
- **Leaflet & React-Leaflet** (for maps)
- **Lucide-React** (for modern, lightweight icons)

## Getting Started
1. Install dependencies:
   ```bash
   npm install
   ```
2. Run the development server:
   ```bash
   npm run dev
   ```
3. Build for production:
   ```bash
   npm run build
   ```
